# bot module

Strategy trading bot tools. Requires API key with **Read + Trade** permissions.

> **Demo mode**: Grid endpoints are supported in demo. DCA, Contract DCA, and Recurring may have limited demo support.

---

## Grid tools (20)

| Tool | Description |
|------|-------------|
| `grid_get_orders` | List active or historical grid bots |
| `grid_get_order_details` | Get details of a specific grid bot |
| `grid_get_sub_orders` | List sub-orders (filled or live) within a grid bot |
| `grid_get_positions` | Get open position for a contract grid bot |
| `grid_get_ai_param` | Get AI-recommended parameters (price range, grid count, APY) |
| `grid_compute_min_investment` | Compute minimum investment required for given grid parameters (public) |
| `grid_rsi_back_testing` | Run RSI back-testing to evaluate trigger conditions (public) |
| `grid_max_quantity` | Get max allowed grid levels for instrument and price range (public) |
| `grid_liquidate_price` | Estimate liquidation price for a contract grid before placing |
| `grid_create_order` | Create a new grid bot (spot or contract) |
| `grid_amend_tpsl` | Modify take-profit / stop-loss prices |
| `grid_amend_basic_param` | Modify price range and grid count of a running grid |
| `grid_instant_trigger` | Immediately trigger a pending grid strategy |
| `grid_adjust_investment` | Add investment to a running grid |
| `grid_withdraw_income` | Withdraw accumulated profits without stopping the bot |
| `grid_close_position` | Close open position for a contract grid |
| `grid_cancel_close_order` | Cancel a pending limit close-position order |
| `grid_compute_margin_balance` | Simulate adding/reducing margin (preview only) |
| `grid_adjust_margin_balance` | Actually add or reduce margin for a contract grid |
| `grid_stop_order` | Stop a running grid bot |

## Spot DCA tools (7)

| Tool | Description |
|------|-------------|
| `dca_get_list` | List active spot DCA strategies |
| `dca_get_history` | List historical spot DCA strategies |
| `dca_get_details` | Get details of a single spot DCA strategy |
| `dca_get_sub_orders` | List sub-orders generated by a spot DCA |
| `dca_get_ai_param` | Get AI-recommended DCA parameters |
| `dca_create` | Create a spot DCA (Martingale) bot |
| `dca_stop` | Stop a spot DCA strategy |

## Contract DCA tools (12)

| Tool | Description |
|------|-------------|
| `contract_dca_get_list` | List active contract DCA strategies |
| `contract_dca_get_history` | List historical contract DCA strategies |
| `contract_dca_get_orders` | List orders for a cycle (requires `cycleId` from `contract_dca_get_cycles`) |
| `contract_dca_get_cycles` | List cycle history (each buy cycle and result) |
| `contract_dca_get_positions` | Get current position for a contract DCA |
| `contract_dca_create` | Create a contract DCA (Martingale) bot |
| `contract_dca_stop` | Stop a contract DCA strategy |
| `contract_dca_manual_buy` | Manually trigger an extra buy outside the regular cycle |
| `contract_dca_margin_add` | Add margin to increase collateral |
| `contract_dca_margin_reduce` | Reduce margin / withdraw collateral |
| `contract_dca_set_take_profit` | Set take-profit price (`tpPrice`) |
| `contract_dca_set_reinvestment` | Enable / disable auto-reinvestment of profits (`allowReinvest`) |

## Recurring Buy tools (7)

| Tool | Description |
|------|-------------|
| `recurring_get_list` | List active recurring buy strategies |
| `recurring_get_history` | List historical recurring buy strategies |
| `recurring_get_details` | Get details of a single recurring buy strategy |
| `recurring_get_sub_orders` | List individual buy orders generated by a recurring strategy |
| `recurring_create` | Create a recurring buy (DCA) strategy |
| `recurring_amend` | Rename a running recurring buy strategy (`stgyName`) |
| `recurring_stop` | Stop a recurring buy strategy |

## CLI

All bot types (Grid, Spot DCA, Contract DCA, Recurring Buy) are supported by the `okx` CLI.

```bash
# ── Spot Grid (algoOrdType: grid) ─────────────────────────────────────────────
okx bot grid orders --algoOrdType grid
okx bot grid orders --algoOrdType grid --history
okx bot grid details --algoOrdType grid --algoId <id>
okx bot grid sub-orders --algoOrdType grid --algoId <id>

# Invest in quote currency (USDT)
okx bot grid create --instId BTC-USDT --algoOrdType grid \
  --maxPx 77000 --minPx 63000 --gridNum 10 --quoteSz 100

# Invest in base currency (BTC)
okx bot grid create --instId BTC-USDT --algoOrdType grid \
  --maxPx 77000 --minPx 63000 --gridNum 10 --baseSz 0.001

# Stop (stopType: 1=sell all, 2=keep holdings)
okx bot grid stop --algoId <id> --algoOrdType grid --instId BTC-USDT --stopType 2

# ── Contract Grid (algoOrdType: contract_grid) ────────────────────────────────
okx bot grid orders --algoOrdType contract_grid
okx bot grid orders --algoOrdType contract_grid --history
okx bot grid details --algoOrdType contract_grid --algoId <id>
okx bot grid sub-orders --algoOrdType contract_grid --algoId <id>

# Neutral direction, 3x leverage, 100 USDT margin
okx bot grid create --instId BTC-USDT-SWAP --algoOrdType contract_grid \
  --maxPx 77000 --minPx 63000 --gridNum 10 \
  --direction neutral --lever 3 --sz 100

# Long direction, 5x leverage
okx bot grid create --instId BTC-USDT-SWAP --algoOrdType contract_grid \
  --maxPx 77000 --minPx 63000 --gridNum 10 \
  --direction long --lever 5 --sz 100

# Stop (stopType: 1=close position+stop, 2=keep position+stop)
okx bot grid stop --algoId <id> --algoOrdType contract_grid --instId BTC-USDT-SWAP --stopType 1

# ── Spot DCA ──────────────────────────────────────────────────────────────────
okx bot dca orders
okx bot dca orders --history
okx bot dca details --algoId <id>
okx bot dca sub-orders --algoId <id>          # filled sub-orders
okx bot dca sub-orders --algoId <id> --live   # pending sub-orders
okx bot dca ai-param --instId BTC-USDT --userRiskMode moderate

# Create spot DCA bot
okx bot dca create \
  --instId BTC-USDT \
  --initOrdAmt 50 --safetyOrdAmt 30 --maxSafetyOrds 3 \
  --pxSteps 0.05 --pxStepsMult 1 --volMult 1 \
  --tpPct 0.03 --slPct 0.20

# Stop spot DCA (stopType: 1=sell holdings, 2=keep holdings)
okx bot dca stop --algoId <id> --instId BTC-USDT --stopType 2

# ── Contract DCA ──────────────────────────────────────────────────────────────
okx bot contract-dca list
okx bot contract-dca list --history
okx bot contract-dca positions --algoId <id>
okx bot contract-dca cycles --algoId <id>
okx bot contract-dca orders --algoId <id> --cycleId <cycleId>

# Create contract DCA bot
okx bot contract-dca create \
  --instId BTC-USDT-SWAP --lever 3 --side buy \
  --initOrdAmt 50 --safetyOrdAmt 30 --maxSafetyOrds 3 \
  --pxSteps 0.02 --pxStepsMult 1 --volMult 1 --tpPct 0.02

okx bot contract-dca stop --algoId <id>
okx bot contract-dca manual-buy --algoId <id> --amt 30
okx bot contract-dca margin-add --algoId <id> --amt 20
okx bot contract-dca margin-reduce --algoId <id> --amt 10
okx bot contract-dca set-tp --algoId <id> --tpPrice 31500
okx bot contract-dca set-reinvest --algoId <id> --allowReinvest true

# ── Recurring Buy ─────────────────────────────────────────────────────────────
okx bot recurring orders
okx bot recurring orders --history
okx bot recurring details --algoId <id>
okx bot recurring sub-orders --algoId <id>

# Create daily recurring buy (100% BTC, 10 USDT/day at 9am UTC+8)
okx bot recurring create \
  --stgyName "Daily BTC" \
  --recurringList '[{"ccy":"BTC","ratio":"1"}]' \
  --amt 10 --period daily --recurringTime 9 --timeZone 8

# Create weekly recurring buy (60% BTC + 40% ETH, every Monday)
okx bot recurring create \
  --stgyName "Weekly Portfolio" \
  --recurringList '[{"ccy":"BTC","ratio":"0.6"},{"ccy":"ETH","ratio":"0.4"}]' \
  --amt 100 --period weekly --recurringDay 1

okx bot recurring amend --algoId <id> --stgyName "New Name"
okx bot recurring stop --algoId <id>
```

# bot 模块

策略交易机器人工具。需要 API Key，开启 **读取 + 交易** 权限。

> **模拟盘**：网格接口支持模拟盘。DCA、合约 DCA、定期投资的模拟盘支持有限。

---

## 网格工具（20 个）

| 工具 | 说明 |
|------|------|
| `grid_get_orders` | 查询运行中或历史网格机器人列表 |
| `grid_get_order_details` | 查询指定机器人详情 |
| `grid_get_sub_orders` | 查询子订单（已成交或挂单中） |
| `grid_get_positions` | 查询合约网格当前持仓 |
| `grid_get_ai_param` | 获取 AI 推荐参数（价格区间、格数、预期年化） |
| `grid_compute_min_investment` | 计算给定参数所需的最小投资数量（公共接口） |
| `grid_rsi_back_testing` | 运行 RSI 回测评估触发条件命中次数（公共接口） |
| `grid_max_quantity` | 获取当前品种和价格区间允许的最大网格数（公共接口） |
| `grid_liquidate_price` | 查询合约网格预估强平价格（下单前风险评估） |
| `grid_create_order` | 创建网格机器人（现货或合约） |
| `grid_amend_tpsl` | 修改止盈止损价格 |
| `grid_amend_basic_param` | 修改运行中网格的价格区间和格数 |
| `grid_instant_trigger` | 立即触发等待中的网格策略 |
| `grid_adjust_investment` | 追加网格投资金额 |
| `grid_withdraw_income` | 提取已实现利润（不停止机器人） |
| `grid_close_position` | 平仓合约网格持仓 |
| `grid_cancel_close_order` | 撤销平仓限价单 |
| `grid_compute_margin_balance` | 模拟计算追加/减少保证金的效果 |
| `grid_adjust_margin_balance` | 实际调整合约网格保证金 |
| `grid_stop_order` | 停止运行中的网格机器人 |

## 现货 DCA 工具（7 个）

| 工具 | 说明 |
|------|------|
| `dca_get_list` | 查询运行中的现货 DCA 策略 |
| `dca_get_history` | 查询历史现货 DCA 策略 |
| `dca_get_details` | 查询单个现货 DCA 详情 |
| `dca_get_sub_orders` | 查询 DCA 子订单 |
| `dca_get_ai_param` | 获取 AI 推荐的 DCA 参数 |
| `dca_create` | 创建现货 DCA（马丁格尔）策略 |
| `dca_stop` | 停止现货 DCA 策略 |

## 合约 DCA 工具（12 个）

| 工具 | 说明 |
|------|------|
| `contract_dca_get_list` | 查询运行中的合约 DCA 策略 |
| `contract_dca_get_history` | 查询历史合约 DCA 策略 |
| `contract_dca_get_orders` | 查询合约 DCA 生成的订单（需传 `cycleId`，从 `contract_dca_get_cycles` 获取） |
| `contract_dca_get_cycles` | 查询每个补仓周期的历史 |
| `contract_dca_get_positions` | 查询合约 DCA 当前持仓 |
| `contract_dca_create` | 创建合约 DCA（马丁格尔）策略 |
| `contract_dca_stop` | 停止合约 DCA 策略 |
| `contract_dca_manual_buy` | 在周期外手动补仓 |
| `contract_dca_margin_add` | 追加保证金 |
| `contract_dca_margin_reduce` | 减少保证金 |
| `contract_dca_set_take_profit` | 设置止盈价格（`tpPrice`，绝对价格） |
| `contract_dca_set_reinvestment` | 开启/关闭利润自动再投资（`allowReinvest`） |

| `recurring_get_details` | 查询单个定期投资详情 |
| `recurring_get_sub_orders` | 查询定期投资生成的子订单 |
| `recurring_create` | 创建定期投资策略 |

## CLI

网格、现货 DCA、合约 DCA、定期投资均支持 `okx` CLI。

```bash
# ── 现货网格（algoOrdType: grid）─────────────────────────────────────────────
okx bot grid orders --algoOrdType grid
okx bot grid orders --algoOrdType grid --history
okx bot grid details --algoOrdType grid --algoId <id>
okx bot grid sub-orders --algoOrdType grid --algoId <id>

# 用计价货币（USDT）投入
okx bot grid create --instId BTC-USDT --algoOrdType grid \
  --maxPx 77000 --minPx 63000 --gridNum 10 --quoteSz 100

# 用基础货币（BTC）投入
okx bot grid create --instId BTC-USDT --algoOrdType grid \
  --maxPx 77000 --minPx 63000 --gridNum 10 --baseSz 0.001

# 停止（stopType: 1=卖出全部持仓, 2=保留持仓）
okx bot grid stop --algoId <id> --algoOrdType grid --instId BTC-USDT --stopType 2

# ── 合约网格（algoOrdType: contract_grid）────────────────────────────────────
okx bot grid orders --algoOrdType contract_grid
okx bot grid orders --algoOrdType contract_grid --history
okx bot grid details --algoOrdType contract_grid --algoId <id>
okx bot grid sub-orders --algoOrdType contract_grid --algoId <id>

# neutral 方向，3倍杠杆，100 USDT 保证金
okx bot grid create --instId BTC-USDT-SWAP --algoOrdType contract_grid \
  --maxPx 77000 --minPx 63000 --gridNum 10 \
  --direction neutral --lever 3 --sz 100

# 做多，5倍杠杆
okx bot grid create --instId BTC-USDT-SWAP --algoOrdType contract_grid \
  --maxPx 77000 --minPx 63000 --gridNum 10 \
  --direction long --lever 5 --sz 100

# 停止（stopType: 1=平仓并停止, 2=保留仓位并停止）
okx bot grid stop --algoId <id> --algoOrdType contract_grid --instId BTC-USDT-SWAP --stopType 1

# ── 现货 DCA ──────────────────────────────────────────────────────────────────
okx bot dca orders
okx bot dca orders --history
okx bot dca details --algoId <id>
okx bot dca sub-orders --algoId <id>          # 已成交子订单
okx bot dca sub-orders --algoId <id> --live   # 挂单中子订单
okx bot dca ai-param --instId BTC-USDT --userRiskMode moderate

# 创建现货 DCA 机器人

  --initOrdAmt 50 --safetyOrdAmt 30 --maxSafetyOrds 3 \
  --pxSteps 0.05 --pxStepsMult 1 --volMult 1 \
  --tpPct 0.03 --slPct 0.20

# 停止现货 DCA（stopType: 1=卖出持仓, 2=保留持仓）
okx bot dca stop --algoId <id> --instId BTC-USDT --stopType 2

# ── 合约 DCA ──────────────────────────────────────────────────────────────────
okx bot contract-dca list

okx bot contract-dca positions --algoId <id>
okx bot contract-dca cycles --algoId <id>
okx bot contract-dca orders --algoId <id> --cycleId <cycleId>

okx bot contract-dca manual-buy --algoId <id> --amt 30
```
